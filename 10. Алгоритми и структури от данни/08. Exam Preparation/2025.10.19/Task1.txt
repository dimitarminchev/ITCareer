# Овчарят, който заспа на p-тата овца
След дълъг ден овчарят Ники не може да заспи. Той решава да брои овцете, но не както обикновено — а да си представи всички възможни редове, в които овцете могат да минат през портата. В стадото има n овце, номерирани от 1 до n. Ники иска да си представи всяка възможна подредба (пермутация) на овцете — в реда, в който те биха прескочили оградата. За да не заспи прекалено бързо, Ники решава да ги номерира: първата подредба е №1, след това №2, и т.н. Напишете програма, която да помогне на Ники да заспи.

## Вход
Програмата прочита едно цяло число - n (брой на овцете)

## Изход
Изведете пермутациите на всички овце, като всяка пермутация се извежда на отделен ред, а в началото на реда се показва поредния номера на пермутацията, последван от двуеточие и разстояние, а след това следва и самата пермутация, чиито елементи са разделени от интервал.

## Примери

| Input | Output   | Comment                       |
| ----- | -------- | ----------------------------- |
| 3     | 1: 1 2 3 | Всички подредби на 3 овце са: |
|       | 2: 1 3 2 | 1: 1 2 3                      |
|       | 3: 2 1 3 | 2: 1 3 2                      |
|       | 4: 2 3 1 | 3: 2 1 3                      |
|       | 5: 3 1 2 | 4: 2 3 1                      |
|       | 6: 3 2 1 | 5: 3 1 2                      |
|       |          | 6: 3 2 1                      |
