# Каски
Организатор на голямо спортно събитие трябва да осигури каски за всички доброволци, които ще помагат на терена.
Всеки доброволец има минимален размер каска, която му пасва.
Наличните каски са с различни размери, но всяка каска може да бъде дадена само на един доброволец.

Организаторът иска да оборудва възможно най-много доброволци с подходящи каски, като спазва следното правило:

Доброволец може да получи каска, само ако размерът ѝ е поне толкова голям, колкото е необходимият за него размер.


## Вход
Програмата прочита два реда:
Първият ред съдържа цели числа — минималните размери каски, нужни за всеки доброволец.
Вторият ред съдържа цели числа — наличните размери каски.

## Изход
Изведете едно цяло число — максималният брой доброволци, които могат да получат подходяща каска.

## Примери

| Вход      | Изход | Обяснение                         |
| --------- | ----- | --------------------------------- |
| 2 3 4 5   | 3     | Доброволец 1 -> каска 2           |
| 1 2 3 3 5 |       | Доброволец 2 -> каска 3           |
|           |       | Доброволец 3 -> няма подходяща    |
|           |       | Доброволец 4 -> каска 5           |
|           |       | Общо 3 доброволци получават каски |
